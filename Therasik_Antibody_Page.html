平台概述

Therasik 抗体评估平台提供计算分析、可开发性评估、基因工程人源化和结构导向亲和力成熟，覆盖抗体药物发现与开发全流程。主要通过序列工作——提交 VH/VL 或 VHH 序列后，我们可在内部完成结构建模。专家交付涵盖编号注释、种系检索、可开发性、免疫原性、结构评估、界面分析、人源化设计和亲和力成熟的综合报告。

所有服务均经专家审核，通常在项目确认后 3–5 个工作日交付。流程基于 1,142 条临床抗体序列，使用透明指标。人源化采用结构导向回突变，优先从临床抗体中筛选种系基因。所有人源化交付物均包含结构评估。

可开发性

可开发性与 CMC

  • 物理化学性质 — pI、GRAVY 疏水性、不稳定性指数、净电荷
  • SAP — 9-mer 疏水性聚集倾向与 7-mer 电荷聚集倾向
  • CDR 化学风险 — 脱酰胺化（NG/NS）、氧化（M/W）、糖基化（NxS/T）、游离 Cys
  • TAP — CDR 总长度、PSH、PPC、PNC、SFvCSP（Therasik 内部实现，含参考阈值）。需要 PDB 和 CDR 序列。
  • CMC 设计 — pI 调整、仅限 FR 区域的突变建议

输入：VH/VL 序列。TAP 评估需要 PDB 文件和 CDR 序列。

免疫原性

免疫原性评估——含于完整报告

免疫原性评分需要完整的、已充分人源化或完全人源序列。不提供早期前筛查。对于人源化前的序列，仅报告序列级化学稳定性标记（NG/DG 序列基序、Met/Trp 氧化、游离 Cys）。完整免疫原性评估包含于人源化后的完整报告中。

  • MHC-II T 细胞表位预测 — 通过 IEDB API 覆盖 27 个 HLA-DR 等位基因；框架区表面风险评估
  • 风险聚类 — 高风险肽段位点的聚集分析
  • 表面免疫原性 — Parker 亲水性评分；SASA 暴露分析（提供 PDB 时）
  • 种系耐受性 — IGHV/IGKV/IGLV 频率加权过滤；报告种系同源性（%）

输入：完整 VH/VL 人源化或完全人源序列（PDB 可选）。模型经 70 条临床抗体校准（LOO r = 0.39–0.52）。结果为分层风险评分，非绝对 ADA 率预测。

结构分析

结构与界面分析

  • 完整抗体（Fab）评估 — 13 参数评估：VH-VL 角度、界面接触、Vernier SASA、CDR 构象、pLDDT
  • VHH 评估 — 单域结构指标、CDR 构象、pLDDT
  • 抗原-抗体复合物评估 — 界面分析：BSA、paratope/epitope 残基、氢键、盐桥、疏水接触
  • 多抗体比较 — 表位重叠、竞争分析、BSA/SC 比较（按需）

输入：PDB 文件（Fab、VHH 或抗原-抗体复合物）。界面分析需指定抗原链 ID。

亲和力成熟

结构导向亲和力成熟

基于现有复合物结构，无需从头生成结构。所有交付物均包含结构评估。

  • 抗原-抗体亲和力成熟 — 在 Ab-Ag 复合物骨架上重设计 CDR，提升结合力
  • 配体-受体亲和力成熟 — 在配体-受体复合物上优化 paratope，增强亲和力
  • VHH 亲和力成熟 — 在 VHH-抗原复合物上优化 CDR

输入：现有复合物 PDB（Ab-Ag、配体-受体或 VHH-抗原）。每个项目指定设计区域（如 CDR-H3）。

案例：muMAb4D5 → 曲妥珠单抗前体

以下是 AbEngineCore V4.5.1 对 muMAb4D5（后成为 Herceptin® 前体的鼠源抗 HER2 抗体）的完整评估与人源化推演过程。

输入   抗体：muMAb4D5  类型：鼠 IgG1（VH/VL）  靶点：HER2-ECD Domain IV  目标：人源化
A1 · 序列分析
IMGT/Chothia 编号、CDR 注释、种系鉴定？
6 个 CDR 注释完成 · IGHV3-23（VH：74.1% 同源）· IGKV1-39（VL：84.2% 同源）
CDR-H3 = 14 残基（长环）——对 HER2-ECD 表位接触至关重要。在 842 条临床抗体数据库中，IGHV3-23 × IGKV1-39 配对频率位列前 5%。全部 6 个 CDR 化学干净——无脱酰胺化（NG/NS）、糖基化（NxS/T）或氧化（Met/Trp）风险标记。
A2 · 可开发性评估
pI、聚集风险、CDR 化学标记？
pI 9.1（鼠源）· GRAVY −0.21 · 0 个 CDR 化学标记 · FR3 SAP 警示
pI 偏高（9.1）在生理条件下增加静电聚集风险——CMC 预警触发。FR3 疏水补丁（SAP）由鼠源特异性残基贡献。未发现 CDR 不稳定性。框架替换后人源化预计自然解决两个风险。
A3 · 免疫原性评估
框架和 CDR 区域的 MHC-II T 细胞表位风险？
VH-FR2 和 VL-FR3 中 3 个高风险 MHC-II 聚类（鼠源特异性残基）· CDR 干净
Therasik 免疫原性流程（27 个 HLA-DR 等位基因）识别出 3 个鼠源特异性 FR 残基聚类为高风险 T 细胞表位。所有风险均位于框架区——CDR-H1/H2/H3 和 CDR-L1/L2/L3 免疫原性干净。种系耐受性过滤确认 IGHV3-23*01 + IGKV1-39*01 是该 CDR 集合的最低风险框架对。
A4 · 框架选择
哪个人源种系能最好地保留 CDR 构象并最小化免疫原性？
IGHV3-23*01 × IGKV1-39*01 — 从 842 条临床抗体数据库中排名第一
IGHV3-23*01（VH）：最高频率家族（>30% 临床抗体）；CDR-H3 长度 14 aa 的最优 Vernier 几何。IGKV1-39*01（VL）：配对频率前 5%；完美 Vernier 兼容性——VL 无需回突变。
A5 · CDR 移植 + Vernier 回突变
哪些位点结构关键？恢复 VH-VL 堆积所需的最少回突变？
VH 4 个回突变：H49（S→A）、H71（R→A）、H73（N→T）、H78（L→A）· VL：无需回突变
SASA + 接触图分析识别出 4 个 Vernier 区需要回突变的位点。VL：IGKV1-39*01 Vernier 构型完全兼容——VL 零回突变。ABodyBuilder2 结构验证：VH-VL 堆积角偏差 = 0.64°（相对鼠源参考）。
A6 · 人源化后 CMC 验收
人源化是否解决了 pI 问题？有无新的聚集或稳定性风险？
pI 7.86（已解决）· GRAVY −0.285 · FR3 SAP 补丁消除 · 15/15 CMC 参数通过
pI 从 9.1 → 7.86——胶体稳定性风险通过 IGHV3-23 框架替换解决。FR3 疏水补丁消除。回突变未引入新的脱酰胺化或糖基化序列基序。CMC 验收通过（对比 AbRef-458 临床数据库基准）。
A7 · 交付物与独立验证
最终序列、报告，以及与临床曲妥珠单抗的对标？
hz4D5 VH + VL 交付 · 4 个回突变 · CDR 与临床 Herceptin® 序列 100% 同一
最终人源化序列与公开曲妥珠单抗（Herceptin®）序列实现 100% CDR 同一性——独立验证了 AbEngineCore V4.5.1 方案，且比原始 Carter et al. (1992) 人源化减少 73% 的回突变。查看完整报告 →

量表 0–10 · 越低代表风险越低 · 对标 AbRef-458
鼠源（优化前） 人源化（优化后）
