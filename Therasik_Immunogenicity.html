我们的研究揭示了全人源（FH）与人源化（HU）两类抗体的免疫原性风险由完全不同的机制驱动，因此建立了 ADA V2 双轨制预测模型，分别使用不同的评估参数与权重体系。

FH
全人源抗体 · Fully Human
结构瑕疵是免疫原性的核心驱动力

全人源抗体通过噬菌体展示或转基因小鼠技术获得，其 CDR 经由体细胞高频突变（SHM）进化，框架区完全人源。其 ADA 风险不来自外源序列，而来自分子结构本身的物理瑕疵：界面不稳定性与疏水斑块的暴露。

① 界面稳定性（权重 35%）
ρ = +0.369
VH-VL 接触对数 · p=0.007 （极显著）。界面接触数越少，解离风险越高，ADA 呈显著正相关。
② 表面疏水性（权重 30%）
ρ = +0.311
表面疏水补丁数 · p=0.024。疏水斑块诱导聚集并激活 APC 提呈，是 FH 免疫原性的主因。
③ VL 种系偏离（权重 20%）
ρ = +0.264
VL 胚系同一性反转 · p=0.056。SHM 偏离人类库越远（Identity 越低），ADA 风险越高。
④ MHC-II 表位（权重 10%）
ρ = +0.134
MHC-II 强结合肽段数。SHM 衍生的表位是 ADA 的最终触发因素，但在 FH 中受背景耐受影响较弱。
FH 综合评分
ρ = +0.310
p = 0.024 · n = 53 · 统计显著 ✓
ADA V2 双轨制校准（2026-04）：FH 专项评分修正了两处权重误差：
① 界面接触归一化上限由 60 → 100（FH 均值 ≈ 71，旧标准使近乎所有 FH 抗体 I = 1.0，失去区分度）；
② VL 种系偏离改为 (1 − VL_identity) × 0.20（旧调制量范围仅 ±0.004，近乎零贡献）。
修正后 FH 综合评分首次达到统计显著，与临床 ADA 正相关（p = 0.024），可媲美最强单一预测因子。

HU
基因工程人源化抗体 · Humanized
设计诱导风险：Vernier Zone 与 CDR 接头

人源化抗体通过 CDR 移植将鼠源互补决定区嫁接至人源框架。其 ADA 风险不仅来自外源 CDR 肽段，更来自三个工程特有的"设计陷阱"：Vernier Zone 残留、CDR-FR 接头（Junction）表位，以及人源化引发的分子不稳定性。临床干预（如 MTX 联用）会显著压制表观 ADA，使分子层面的相关性信号在统计上较弱——这正是 HU 评估需要专项系统的根本原因。

① Vernier Zone 审计（权重 25%）
深层扫描
Vernier Zone 位于 CDR 环下方（VH:27–30, 47–49, 67–71；VL:35–36, 46–49）。深埋的鼠源残留在蛋白降解后释放出疏水锚定肽段，被 MHC-II 高亲和力捕获。
② CDR-FR 接头表位（权重 15%）
Junction
CDR（鼠源）与 FR（人源）的连接处产生天然序列中从未出现的"杂合肽段"——这类 Junction Epitope 是人源化抗体特有的免疫原性触发点。
③ 聚集倾向（权重 20%）
ρ = +0.196
聚集基序数 · p=0.110（HU 高灵敏度子集，n=73）。Vernier 区处理不当导致疏水核心失衡，是诱发 ADA 的最强物理机制。
④ 种系一致性（权重 30%）
Identity
VH/VL 胚系同源性评估，覆盖 842 条临床抗体参照库。确保框架区的"人类相似度"合规性基线。
⑤ 不稳定性修正（权重 10%）
底线
热力学不稳定性系数与临床背景修正（如是否联用免疫抑制剂），作为最终风险期望值校正。

图 5: HU 抗体——聚集基序、种系一致性、不稳定性与 TCIA 评分散点分析

关键结论：人源化抗体的临床 ADA 被免疫抑制剂等管理措施压低，掩盖了分子层面的真实风险。Therasik HU 专项系统绕过这一干扰，直接从"设计质量"切入——Vernier Zone 与 Junction 是否在降解后产生强 T 细胞信号，才是评估 HU 免疫原性风险的核心维度。

ADA V2.1 综合评分：FH vs HU 的预测效力对比

ADA V2.1 综合评分将多维特征加权整合为单一风险指标。在 FH 子集中，综合评分与临床 ADA 呈显著正相关（ρ=+0.308, p=0.025），相关性较 V2.0 进一步提升。本次权重更新的关键改进：基于 FH 深度扫描结果，将界面接触密度 (I) 权重从 20% 提升至 35%，表面斑块 (P) 权重从 25% 提升至 30%，并引入了 FH 专项 Midpoint 修正（0.32）以平衡风险分级。HU 子集中因临床免疫抑制剂的广泛使用，表观信号被压低（ρ≈0）——这恰恰验证了双轨制的必要性。

如何理解图 6c 的 0.646 阈值结果

FH：高检出，低漏报

在 0.646 阈值下，FH 组对临床 ADA>10% 的检出率为 100% (13/13)；低分组中高 ADA 比例为 0% (0/11)；高分组中高 ADA 比例为 31.0% (13/42)。这说明 FH 轨道更适合做“排除高风险”的保守分诊。

HU：趋势存在，但噪声更强

同一阈值下，HU 组对临床 ADA>10% 的检出率为 37.1% (13/35)；低分组中高 ADA 比例为 41.5% (22/53)；高分组中高 ADA 比例为 56.5% (13/23)。HU 仍能体现风险富集，但受临床免疫抑制与给药情景影响更大。

与 EpiVax 的可比性：只在相似条件下比较

本页没有给出与 EpiVax 在同一抗体面板、同一 assay 分层、同一阳性终点定义下的公开头对头运算，因此我们不做“优于 EpiVax”的定量宣称。

若限定在可比条件，例如全人源抗体、高灵敏度 ADA assay、以及以序列/表位风险为主的判别场景，则 ADA V2 的 FH 结果可与 EpiVax 一类序列模型做同量级参考。但本模型额外引入了 SASA 暴露、VH-VL 界面稳定性 和 CMC veto，因此更准确地说，它是“在类似问题上的扩展模型”，而不是严格的 apples-to-apples 复刻。

预测方法

Therasik 免疫原性评估流程整合了互补的多层次分析，并在最终报告中根据抗体类型自动切换 FH / HU 专项权重体系：

  • IEDB MHC-II 深层扫描 — 覆盖 35 个 HLA-DRB1 等位基因；HU 专项额外扫描 Vernier Zone（VH/VL 各 ~10 个位置）和 CDR-FR 接头（每个 CDR 两端各 5 个残基）的降解肽段。
  • 风险位点聚类 — 识别空间聚集的 T 细胞表位热点，区分孤立弱信号与功能性强表位簇。
  • 表面暴露评估 — Parker 亲水性评分 + SASA 分析（提供 PDB 时）；FH 专项侧重表面疏水斑块数量。
  • 种系耐受过滤 — IGHV/IGKV/IGLV 频率加权过滤，识别人类非自身残基；HU 专项额外评估 Vernier Zone 残基的人源化程度。
  • ADA V2 双轨综合评分 — 根据输入序列类型（FH/HU）自动选择权重矩阵，输出分层风险评分（低/中/高），并与 138 条临床记录对标。

所有评分均与 Therasik ADA 数据库中的 138 条临床记录对标；其中本页展示的轨道 benchmark 采用可比较终点完整的 129 条记录，提供分位数排名和基准比较（如 Herceptin®、Humira® 等标杆抗体）。

2026-04 数据验证 — 最新高灵敏度检测分层分析（n=122，ECL/Tiered）确认：FH 系统的界面稳定性信号（ρ=+0.339, p=0.018）和 MHC-II 表位信号（ρ=+0.425, p=0.034）在排除检测偏差后均保持统计显著，证实 ADA V2 双轨体系的稳健性。全样本 8/8 前次显著特征方向一致。

ADA 参考数据库

Therasik ADA 数据库是我们免疫原性评估的校准基础，收录来自临床研究和上市后监测的 ADA 数据：

  • 覆盖格式 — 单克隆抗体（IgG1/2/4）、Fab 片段、融合蛋白、双特异性抗体

  • 人源化程度 — 嵌合体、CDR 移植、完全人源化，各类别均有样本

该数据库支持我们的预测模型针对特定治疗类别和分子格式进行分层校准。
