技术平台

AbEngineCore — 统一设计引擎

AbEngineCore 是 Therasik 的核心计算框架，深度集成 30+ 种业界领先的生物计算与 AI 建模工具。我们将机器学习算法与 1,142 条临床抗体基准数据有机整合，交付真正可执行的设计方案。以下是工作流中常用的 8 核心技术组件：

结构预测

AlphaFold2

高精度蛋白质结构预测，用于抗体可变区和复合物建模，为可开发性与免疫原性评估提供结构基础。

抗体结构

ABodyBuilder2

专为 Fv 区优化的抗体结构预测模型，CDR 构象精度优于通用结构预测工具。

对接精修

HADDOCK3

基于数据驱动的蛋白质-蛋白质对接与界面精修，用于抗原-抗体复合物分析及亲和力成熟。

序列设计

ProteinMPNN

深度学习驱动的反向折叠序列设计，用于从头 CDR 设计和人源化框架优化。

扩散生成

RFdiffusion

基于扩散模型的蛋白质骨架设计，支持特定表位靶向抗体从头生成。

编号注释

ANARCI

深度学习抗体编号系统，支持 Kabat、IMGT、Chothia 方案，精准划定 FR 与 CDR 区域。

免疫原性

IEDB API

基于 IEDB 的 MHC-II 结合预测，覆盖 35 个 HLA-DRB1 等位基因，用于 T 细胞表位风险评分。

热稳定性

ThermoMPNN

突变热稳定性预测，结合可开发性分析筛选亲和力成熟候选。

能力覆盖

全方位评估能力

抗体评估

  • 序列分析 — Kabat/IMGT 编号、CDR 划定、种系基因归属
  • CMC 可开发性 — pI、疏水性、SAP、TAP、化学修饰风险
  • 免疫原性 — MHC-II 27 等位基因预测、表面风险评估
  • 结构分析 — VH-VL 角度、界面接触、paratope/epitope 分析
  • VH/VL 人源化 — 结构驱动回突变，842 条临床种系优先
  • VHH 人源化 — 表面重塑与框架移植，含结构评估
  • VH → VHH 转化 — CDR 保留骆驼化改造
  • 亲和力成熟 — ΔΔG 扫描、5D 过滤、上位协同分析

细胞治疗与多特异性

  • CAR-T 架构设计 — 结合域、间隔区、跨膜域、共刺激域选择
  • CAR 元件基准 — 237 个临床 CAR 元件数据库比对
  • 双特异性格式筛选 — 134+ 临床双抗格式可开发性基准
  • 接头设计 — 柔性/刚性接头选择与长度优化
  • pI 工程 — 双链电荷不对称性设计，改善分离纯化
  • 疫苗设计 — 新抗原 neoantigen 筛选、异克隆优化、多表位 mRNA
  • 免疫原性数据库 — 138 条 ADA 记录，临床免疫原性率关联分析
